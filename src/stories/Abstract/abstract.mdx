# Abstract

## Overview

The Abstract component displays a short summary or introductory passage of rich-text content, giving users a quick overview of a page before they read the full content.

The content is authored as WYSIWYG markup and sits inside a styled `qld__body` section. The wrapping section can be given an anchor `id` so it can be linked to directly; when none is supplied it falls back to `abstract-{assetid}`.

## Empty state

When no content is supplied, the component still renders its styled wrapper — useful for confirming layout and spacing without content.
