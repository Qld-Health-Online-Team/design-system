# Body

## Overview

The Body component is a section wrapper that holds a region of page content. It applies the `qld__body` styling and lets authors set a background theme and control the width of the content area.

It is the primary content container used to build a page, and can be stacked to create sections with alternating backgrounds.

## Variants

The **background** can be set to one of five themes — White (default), Light, Dark, Alternate or Dark Alternate — to visually separate sections of a page.

The **body width** controls the content area: Full-width spans the container, while Half-width constrains it to a narrower column.

The wrapping `<section>` can be given an anchor `id`; when none is supplied it falls back to `body-{assetid}`.
